# Sport meeting system - REST API

Zastosowane technologie i narzędzia:
- Spring Boot 2.2.0
- Spring Security
- Java 8
- Maven 4.0.0
- Hibernate
- PostgreSQL 12.2
- IntelliJ IDEA 2019.2.3
- pgAdmin 4
